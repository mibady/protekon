# OSHA Publications — Protekon Reference Library

27 source documents extracted and organized for the Protekon compliance SaaS app.
Covers OSHA, Cal/OSHA, HIPAA/HHS, DOT/FMCSA, Cal EPA/DPR, and CA civil law.

## Directory Structure

```
OSHA Publications/
├── pdfs/                                      ← 24 source files (PDFs + 1 .txt)
├── Machine Guarding.pptx                      ← Cal/OSHA machine guarding presentation
├── Silica-exposure-control-model.docx         ← OSHA model silica plan (29 CFR 1926.1153)
├── extracted/                                 ← Structured data extracted from all sources
│   ├── extracted-data.md                      ← Original 16 OSHA publications
```
